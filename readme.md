# Модуль шаблонов проектов Bitrix24

## Установка
1. Скопировать репозиторий
2. Перенести модуль на свою версию Bitrix
3. Marketplace -> Установленные решения -> Project Templates -> Установить

Для работы модуля необходимо добавить пункт меню:
* Название: Шаблоны проектов 
* Ссылка: /project-templates/

Модуль содержит в себе две новые таблицы БД: vendor_project_templates и vendor_project_template_tasks
С помощью данного модуля можно создавать свои шаблоны проектов, задавая им список задач, названия, а также ответственных за проект в целом и за каждую подзадачу.
